Deployment of a Medical imaging web server

In this project we introduce a medical imaging web service based on Orthanc, an open-source project of DICOM server, and a pipeline to deploy such server automatically and consistently.
This little project involves a number of technical deets in DevOps, to deliver a web application prototype with an automated deployment pipeline.
Yi will introduce this project and do demo.
